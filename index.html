üíª How I made this website

I made an entire page about how I made the website and what I used. You can find it in the "Behind the Scenes" button in the navbar. Anyways, here's a small recap:

I used Visual Studio Code as my code editor, and I took the icons from a website called icons8. Feel free to take a look there, it's a great website with lots of icons. To use them for free you just need to insert a link in your website.

The image on the left is a screenshot of me programming the website, while it wasn't even completed yet. This website took a few weeks to make. The initial version was different from the final one.

I tried to make the best design possible while not making it too complicated, and I think I succeded at this task! I hope you like this website as much as me, but, if you want to give me some suggestions or report a bug, contact me or write an issue on this website's GitHub repository.
